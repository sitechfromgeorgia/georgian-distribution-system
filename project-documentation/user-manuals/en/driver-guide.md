## Managing Availability

### Going Online

1. Toggle **Available** switch
2. System marks you as available
3. You can receive new assignments

**Status**: Green (Available)

### Going Offline

1. Toggle **Available** switch off
2. Complete any active deliveries first
3. No new assignments received

**Status**: Gray (Offline)

### Auto-Offline

System automatically marks you offline:
- After 30 minutes of inactivity
- If you miss 3 consecutive assignments
- During scheduled maintenance

**Re-enable availability manually**

## Delivery Process

### Step-by-Step Workflow

#### Step 1: Pick Up Order

1. Navigate to restaurant
2. Arrive at pickup location
3. Find order using order number
4. Verify items with restaurant
5. Tap **Mark as Picked Up**
6. Status changes to **Picked Up**

**Checklist**:
- [ ] All items present
- [ ] Items properly packed
- [ ] Special handling noted
- [ ] Restaurant signature (if required)

#### Step 2: In Transit

1. Start navigation to delivery address
2. Tap **Start Delivery**
3. Status changes to **In Transit**
4. Customer can track your location

**En Route**:
- Follow GPS directions
- Drive safely
- Keep items secure
- Update customer if delayed

#### Step 3: Arrive at Delivery

1. Park safely
2. Collect items
3. Find customer's location
4. Call if needed: Tap phone icon

#### Step 4: Hand Over Order

1. Verify delivery address
2. Hand items to customer
3. Confirm customer name
4. Get signature (if required)
5. Tap **Mark as Delivered**

#### Step 5: Complete Delivery

1. Confirm delivery details
2. Add notes if needed
3. Take photo (if required)
4. Tap **Complete Delivery**
5. Status changes to **Delivered**

## Completing Deliveries

### Successful Delivery

1. Customer receives order
2. Mark as delivered
3. Add delivery notes (optional)
4. Earnings updated immediately

### Delivery Issues

#### Customer Not Available

1. Try calling customer
2. Wait 5 minutes
3. Call support
4. Options:
   - Leave with neighbor
   - Return to restaurant
   - Leave at safe location

**Document with photos!**

#### Wrong Address

1. Contact customer immediately
2. Verify correct address
3. If nearby, deliver to correct location
4. If far, contact support
5. Update address in system

#### Items Missing/Damaged

1. Do not deliver damaged items
2. Photo documentation
3. Contact support immediately
4. Return to restaurant if needed

### Cannot Complete Delivery

1. Tap **Report Issue**
2. Select issue type:
   - Customer unavailable
   - Address incorrect
   - Access issues
   - Vehicle problems
3. Add details
4. Contact support
5. Follow instructions

## Safety Guidelines

### Driving Safety

✅ **DO**:
- Follow traffic laws
- Use seatbelt
- Obey speed limits
- Park legally
- Secure items properly

❌ **DON'T**:
- Use phone while driving
- Rush or speed
- Park illegally
- Drive tired
- Skip breaks

✅ **DO**:
- Stay in well-lit areas
- Keep phone charged
- Share location with family
- Trust your instincts
- Report suspicious activity

❌ **DON'T**:
- Enter unsafe locations
- Deliver alone at night (if uncomfortable)

## FAQ

**Q: When do I get paid?**
A: Weekly payments every Monday for previous week.

**Q: What if order is cancelled?**
A: Partial payment for time/distance if already en route.

**Q: Can I reject assignments?**
A: Yes, but affects acceptance rate. Maintain 80%+.

**Q: What about vehicle expenses?**
A: You're responsible for fuel, maintenance, insurance.

**Q: Do I get tips?**
A: System supports tips. Customers can tip in app.

**Q: What if I have accident?**
A: Stop deliveries, ensure safety, call emergency, report to support.

**Q: Can I work part-time?**
A: Yes! Work flexible hours. Toggle availability as needed.
