2. იხილეთ ყველა თქვენზე დანიშნული შეკვეთა
3. დალაგებული პრიორიტეტის მიხედვით

### შეკვეთის ინფორმაცია

თითოეული შეკვეთა აჩვენებს:
- შეკვეთის ნომერი
- რესტორნის სახელი და მისამართი
- მიწოდების მისამართი
- მომხმარებლის ტელეფონი
- ნივთების რაოდენობა
- სპეციალური ინსტრუქციები
- მანძილი აღების ადგილამდე

5. თუ არ გასცემთ პასუხს, შეკვეთა გადანაწილდება

**პრო რჩევა**: სწრაფად მიიღეთ კარგი რეიტინგის შესანარჩუნებლად!

## მიწოდების პროცესი

### ნაბიჯ-ნაბიჯ სამუშაო პროცედურა

#### ნაბიჯი 1: შეკვეთის აღება

1. ნავიგაცია რესტორნამდე
2. მიხვდით აღების ადგილას
3. იპოვეთ შეკვეთა შეკვეთის ნომრის გამოყენებით
4. გადაამოწმეთ ნივთები რესტორანთან ერთად
5. შეეხეთ **აღებულად მონიშვნა**
6. სტატუსი იცვლება **აღებულად**

**საკონტროლო სია**:
- [ ] ყველა ნივთი წარმოდგენილია
- [ ] ნივთები სათანადოდ არის შეფუთული
- [ ] სპეციალური მიმოქცევა აღნიშნულია
- [ ] რესტორნის ხელმოწერა (თუ საჭიროა)

#### ნაბიჯი 2: გზაში

1. დაიწყეთ ნავიგაცია მიწოდების მისამართამდე
2. შეეხეთ **მიწოდების დაწყება**
3. სტატუსი იცვლება **გზაში**
4. მომხმარებელს შეუძლია თქვენი ადგილმდებარეობის თვალყურის დევნება

**გზაზე**:
- მიჰყევით GPS ინსტრუქციებს
- უსაფრთხოდ იმართეთ
- შეინარჩუნეთ ნივთები უსაფრთხოდ
- განაახლეთ მომხმარებელი თუ დაგიანებთ

#### ნაბიჯი 3: მისამართზე მისვლა

1. უსაფრთხოდ დაიპარკეთ
2. შეაგროვეთ ნივთები
3. იპოვეთ მომხმარებლის ადგილმდებარეობა
4. დარეკეთ საჭიროების შემთხვევაში: შეეხეთ ტელეფონის ხატულას

#### ნაბიჯი 4: შეკვეთის გადაცემა

1. გადაამოწმეთ მიწოდების მისამართი
2. გადასცეთ ნივთები მომხმარებელს
3. დაადასტურეთ მომხმარებლის სახელი
4. მიიღეთ ხელმოწერა (თუ საჭიროა)
5. შეეხეთ **მიტანილად მონიშვნა**

#### ნაბიჯი 5: მიწოდების დასრულება

1. დაადასტურეთ მიწოდების დეტალები
2. დაამატეთ შენიშვნები საჭიროების შემთხვევაში
3. გადაიღეთ ფოტო (თუ საჭიროა)
4. შეეხეთ **მიწოდების დასრულება**
5. სტატუსი იცვლება **მიტანილად**
