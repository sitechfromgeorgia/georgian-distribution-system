## рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

### рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃАрЃћрЃЦрЃфрЃўрЃћрЃЉрЃў

#### ­ЪЊі рЃЊрЃдрЃћрЃЋрЃљрЃюрЃЊрЃћрЃџрЃў рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ

**рЃЊрЃдрЃћрЃЋрЃљрЃюрЃЊрЃћрЃџрЃў рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў**
- рЃЊрЃдрЃћрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃАрЃБрЃџ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў
- рЃЊрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ

**рЃЏрЃЮрЃџрЃЮрЃЊрЃўрЃюрЃерЃў рЃЏрЃДрЃЮрЃцрЃў рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў**
- рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃљрЃА рЃЏрЃЮрЃџрЃЮрЃЊрЃўрЃюрЃерЃў рЃЏрЃДрЃЮрЃцрЃў рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў
- **рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ!**

**рЃЊрЃдрЃћрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў**
- рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЏрЃўрЃбрЃљрЃюрЃўрЃџрЃў рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў
- рЃЊрЃдрЃћрЃЋрЃљрЃюрЃЊрЃћрЃџрЃў рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃћрЃџрЃў

#### ­ЪЊд рЃЉрЃЮрЃџрЃЮ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў

- рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЉрЃЮрЃџрЃЮ 10 рЃерЃћрЃЎрЃЋрЃћрЃЌрЃљрЃА

- рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃћрЃЉрЃў

2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ** рЃдрЃўрЃџрЃљрЃЎрЃА
3. рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ рЃюрЃўрЃЋрЃЌрЃћрЃЉрЃўрЃА рЃцрЃљрЃАрЃћрЃЉрЃў
4. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ**
5. рЃАрЃбрЃљрЃбрЃБрЃАрЃў рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ **рЃњрЃљрЃцрЃљрЃАрЃћрЃЉрЃБрЃџрЃљрЃЊ**

### рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ

1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃерЃћрЃЎрЃЋрЃћрЃЌрЃљрЃА рЃАрЃўрЃљрЃерЃў
2. рЃњрЃљрЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ рЃЊрЃўрЃљрЃџрЃЮрЃњрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА:
   - рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў
   - рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃАрЃбрЃљрЃбрЃБрЃАрЃў
   - рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў
   - рЃДрЃЋрЃћрЃџрЃљ рЃюрЃўрЃЋрЃЌрЃў рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃЌ
   - рЃАрЃБрЃџ рЃЌрЃљрЃюрЃ«рЃљ
   - рЃЏрЃФрЃдрЃЮрЃџрЃўрЃА рЃўрЃюрЃцрЃЮ (рЃЌрЃБ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃўрЃљ)
   - рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ

### рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљ

**рЃЏрЃФрЃдрЃЮрЃџрЃўрЃА рЃЊрЃљрЃюрЃўрЃерЃЋрЃюрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ**:

1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃДрЃБрЃарЃўрЃА рЃЊрЃћрЃЋрЃюрЃћрЃЉрЃљ** рЃдрЃўрЃџрЃљрЃЎрЃА
3. рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃЏрЃФрЃдрЃЮрЃџрЃўрЃА рЃљрЃЊрЃњрЃўрЃџрЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ рЃарЃБрЃЎрЃљрЃќрЃћ
4. рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃљ рЃЏрЃўрЃАрЃЋрЃџрЃўрЃА рЃАрЃљрЃЋрЃљрЃарЃљрЃБрЃЊрЃЮ рЃЊрЃарЃЮ
5. рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ

### рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃљ

**рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃљ рЃЌрЃБ**:
- рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃљ рЃЏрЃЮрЃџрЃЮрЃЊрЃўрЃюрЃерЃў, рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃБрЃџрЃў рЃљрЃю рЃњрЃљрЃцрЃљрЃАрЃћрЃЉрЃБрЃџрЃў
- рЃ»рЃћрЃа рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃдрЃћрЃЉрЃБрЃџрЃў рЃЏрЃФрЃдрЃЮрЃџрЃўрЃА рЃЏрЃўрЃћрЃа

**рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃљрЃБрЃЦрЃЏрЃЮрЃЌ**:

1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃљ** рЃдрЃўрЃџрЃљрЃЎрЃА
3. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃќрЃћрЃќрЃў:
   - рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ
   - рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃюрЃўрЃЋрЃЌрЃћрЃЉрЃў
   - рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
   - рЃАрЃ«рЃЋрЃљ (рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЌ)
4. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃљ**
5. рЃерЃћрЃЎрЃЋрЃћрЃЌрЃљ рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃБрЃџрЃў

**Рџа№ИЈ рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ**: рЃЋрЃћрЃа рЃњрЃљрЃљрЃБрЃЦрЃЏрЃћрЃЉрЃЌ рЃЏрЃФрЃдрЃЮрЃџрЃўрЃА рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃљрЃдрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ!

## рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

### рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ

1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў** рЃЏрЃљрЃарЃфрЃ«рЃћрЃюрЃљ рЃЏрЃ«рЃљрЃарЃћрЃА
2. рЃЊрЃљрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЌ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЎрЃљрЃбрЃљрЃџрЃЮрЃњрЃў
3. рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў рЃцрЃљрЃАрЃћрЃЉрЃўрЃЌ

### рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА:
- рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў
- SKU (рЃАрЃљрЃгрЃДрЃЮрЃЉрЃўрЃА рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃћрЃарЃЌрЃћрЃБрЃџрЃў)
- рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ
- рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃцрЃљрЃАрЃў
- рЃћрЃарЃЌрЃћрЃБрЃџрЃў (рЃЎрЃњ, рЃџрЃўрЃбрЃарЃў, рЃфрЃљрЃџрЃў, рЃЊрЃљ рЃљ.рЃе.)
- рЃЏрЃљрЃарЃљрЃњрЃўрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљ

### рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃФрЃўрЃћрЃЉрЃљ

**рЃАрЃгрЃарЃљрЃцрЃў рЃФрЃўрЃћрЃЉрЃљ**:
1. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃФрЃўрЃћрЃЉрЃўрЃА рЃЋрЃћрЃџрЃў рЃќрЃћрЃЏрЃЮрЃЊрЃљрЃю
2. рЃљрЃЎрЃарЃўрЃцрЃћрЃЌ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃю SKU
3. рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў рЃцрЃўрЃџрЃбрЃарЃЊрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў

**рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ**:
1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ** рЃЕрЃљрЃЏрЃЮрЃерЃџрЃљрЃЊ рЃЏрЃћрЃюрЃўрЃБрЃА
2. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ (рЃарЃФрЃўрЃА рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў, рЃАрЃљрЃцрЃБрЃљрЃарЃўрЃЌ рЃЊрЃљ рЃљ.рЃе.)
3. рЃўрЃ«рЃўрЃџрЃћрЃЌ рЃњрЃљрЃцрЃўрЃџрЃбрЃарЃБрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў

### рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ

рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў:

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ **рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў** рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ** рЃдрЃўрЃџрЃљрЃЎрЃА
3. рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ:
   ```
   рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў: рЃљрЃ«рЃљрЃџрЃў рЃўрЃЮрЃњрЃБрЃарЃбрЃў
   рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ: рЃарЃФрЃўрЃА рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў
   рЃерЃћрЃЏрЃЮрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃАрЃў: 3.50 РѓЙ
   рЃљрЃдрЃгрЃћрЃарЃљ: рЃюрЃљрЃбрЃБрЃарЃљрЃџрЃБрЃарЃў рЃўрЃЮрЃњрЃБрЃарЃбрЃў, 500рЃњ
   ```
4. рЃгрЃљрЃарЃљрЃЊрЃњрЃўрЃюрЃћрЃЌ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ
5. рЃљрЃЊрЃЏрЃўрЃюрЃў рЃњрЃљрЃюрЃўрЃ«рЃўрЃџрЃљрЃЋрЃА рЃЊрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃЌрЃБ рЃЊрЃљрЃЏрЃбрЃЎрЃўрЃфрЃЊрЃћрЃЉрЃљ
